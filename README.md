# ☕ Core Java Fundamentals & Algorithms

> **"A collection of Java programs demonstrating mastery of Object-Oriented Programming (OOP), Data Structures, and Algorithmic Logic."**

### 📂 Repository Overview
This repository serves as a code library for Core Java concepts. It contains solutions to common algorithmic problems, focusing on efficiency, logic building, and the implementation of OOP principles (Inheritance, Polymorphism, Encapsulation).

### 🧠 Key Concepts Implemented

#### 1. Number Theory & Mathematical Logic
* **Special Numbers:** Algorithms to identify Disarium, Duck, Neon, Spy, Happy, and Niven numbers.
* **Basic Math:** Prime number generation (`Prime.java`, `PrimeRan.java`), Factorials, HCF calculation, and Power functions.
* **Geometry:** Area and perimeter logic (`Circle.java`, `Rect.java`, `Tria.java`).

#### 2. String & Array Manipulation
* **String Operations:** Palindrome detection (`Palindrome.java`), Case toggling (`LowerToUpper.java`, `TitleCase.java`), Substring search.
* **Array Processing:** Finding Max/Min values (`ArrayMax.java`), Length calculation, and specific element filtering (`ArrayEven.java`).

#### 3. Object-Oriented Programming (OOP)
* **Inheritance:** Demonstrating parent-child relationships (`Child.java`).
* **Interfaces:** Implementing abstract layers (`NewInterface1.class`).
* **Class Structure:** Modular code organization using custom classes (`MyClass.java`).
